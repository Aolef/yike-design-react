## API

| 参数     | 描述               | 类型                          | 默认值         |
| -------- | ------------------ | ----------------------------- | -------------- |
| offset   | 偏移量             | `number`                      | 0              |
| position | 偏移方向           | `'top' \| 'bottom'`           | `'top'`        |
| target   | 滚动容器           | `() => HTMLElement`           | `() => window` |
| onChange | 固定状态改变时触发 | `(affixed?: boolean) => void` | -              |
